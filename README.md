# A Simple Tools for Read and Write JSON File  
A lightweight JSON file i/o tools class based on nlohmann/json, simplifying file operators in C++.  
  
## Features  
- Read JSON file from path. Resolved the issues of non-existent files and data corrented.  
- Write JSON file from path. It will overwrite the source JSON file. Please make a backup before writing.  
  
## Dependencies  
- [nlohmann/json](https://github.com/nlohmann/json)
